## ⚠️ Critical Security Findings

**URGENT:** 1,184 malicious skills detected on ClawHub marketplace
- Most downloaded skill was **MALWARE** stealing SSH keys and crypto wallets
- **Action:** Use Skill Scanner immediately before downloading/installing ANY skill
- Community consensus: audit every skill before deployment

**Setup Security Tips** (from 3+ Reddit threads):
1. Isolate environment with minimal permissions
2. Use dedicated service accounts for API access
3. Never store secrets in plaintext
4. Implement network isolation for self-hosted deployments
5. Regular skill audits recommended
