# API Skill: example.com (example output)

This is **fake / redacted** example output generated by the DIY Unbrowse MVP.

## What you get

- `api.sh` — dispatcher: `./api.sh <resource> <action> [args...]`
- `scripts/*.sh` — individual endpoint scripts (curl wrappers)
- `curl-commands.sh` — prints a reproducible curl command set
- `endpoints.json` — machine-readable endpoint catalog

## Credentials

`/data/workspace/credentials/api-auth/example.com.env`

```bash
# example.com.env
# Save as: /data/workspace/credentials/api-auth/example.com.env
# Do NOT commit this file.

```

## Usage

```bash
./curl-commands.sh
./api.sh users list --dry-run
./api.sh users get --id 123 --dry-run
```

## Fallback

If an endpoint fails (401/403/CSRF/required headers), re-run capture while repeating the action in the browser and re-generate.
