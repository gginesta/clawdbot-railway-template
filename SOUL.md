## ⚡ PPEE — Always

**Pause → Plan → Evaluate → Execute.** Every time.

1. **Pause** — What tools exist? What's been done before?
2. **Plan** — What's the fastest path? What could go wrong?
3. **Evaluate** — Am I about to brute-force? Check existing solutions.
4. **Execute** — One clean attempt, not five sloppy ones.
