## Target User

**Superconnectors** — people who see themselves as network hubs, relationship builders, connectors of people and opportunities. NOT B2B/VCs/accelerators.

This is prosumer territory, which makes Meta ads viable.

---

## Reference: Meta Ads Kit (@TheMattBerman)

Source: https://x.com/TheMattBerman/status/2027220216409723296

### The System

1. **Daily health check** — `social-cli` wraps Meta Marketing API
2. **Fatigue detection** — frequency > 3.5 = audience cooked, CTR about to drop
3. **Auto-pause bleeders** — CPA > 2.5x target for 48hrs → pause
4. **Auto-scale winners** — shift budget to best performers
5. **Generate ad copy** — analyze winning hooks/angles/CTAs, create variations
6. **Upload directly** — publish to Meta Ads Manager without manual flow

### Skills (free kit)

- `meta-ads` — daily checks + auto-pause
- `ad-creative-monitor` — fatigue detection  
- `budget-optimizer` — efficiency scoring + shift recommendations
- `ad-copy-generator` — writes variations from winners
- `ad-upload` — publishes creative directly

### Key Tool

`social-cli` by @vishalojha_me — wraps Meta Marketing API (token refresh, pagination, rate limits handled)

## Prerequisites Before Deploying

- [ ] 50+ paying customers (understand who converts)
- [ ] Proven messaging from organic conversions
- [ ] Creative assets (video/images) ready
- [ ] Ad budget allocated ($1-5k/mo minimum to learn)
- [ ] Meta Business Manager set up
- [ ] Pixel installed, conversion events configured
- [ ] Leonardo assigned to manage

---

## Implementation Plan (When Ready)

1. **Setup phase**
   - Install social-cli
   - Configure Meta Business Manager
   - Install skills from Meta Ads Kit
   - Set target CPA based on LTV

2. **Creative phase**
   - Create initial ad variations based on organic winners
   - Video testimonials from early customers
   - Hook variations tested organically first

3. **Launch phase**
   - Start with small daily budget ($50-100/day)
   - Let Leonardo monitor and optimize
   - Daily brief to Telegram

4. **Scale phase**
   - Increase budget on winners
   - Leonardo generates new copy variations
   - Continuous fatigue monitoring
