## 4. Site Architecture (Recommended)

Simple, fast, high-signal. One page or three-page max. No blog (Guillermo doesn't write regularly — better to link Medium where it exists than fake consistency).

### Option A: Single Long-Scroll Page (Recommended)
```
[Hero]          — Name + one-liner + photo
[Now]           — What I'm building right now (Brinc / Cerebro / Mana)
[Track Record]  — Portfolio highlights, exits, key deals
[Background]    — Brief bio, Warwick, Infiniti, journey to HK
[Speaking]      — Past + upcoming events
[Connect]       — Email / LinkedIn / X — clean, no form
```

### Option B: Minimal Multi-Page
```
/ (home)        — Hero + Now + Connect CTA
/about          — Background + bio
/work           — Brinc / Cerebro / Mana with logos + links
```

**Recommendation: Option A.** Faster to build, faster to load, lower maintenance.

## 6. Visual Direction

**Tone:** Confident, clean, warm. Not corporate. Not startup-bro.

**References to consider:**
- Paul Graham's site (ultra-minimal, substance over style)
- Patrick Collison's personal site (clean, credibility-forward)
- Naval Ravikant-style: sparse text, weight from substance

**Colors:** Neutral base (white/off-white), one accent. Not Brinc blue (that's a company brand). Something personal — deep green, slate, or warm gray.

**Photo:** Professional but not stiff. Outdoors or casual office works well given Guillermo's active lifestyle brand.

**Typography:** One font family. Clean sans-serif (Inter, Neue Haas Grotesk, or similar).
