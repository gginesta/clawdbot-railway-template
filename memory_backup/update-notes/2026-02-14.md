## 🔧 Fixes (Relevant to Our Setup)

1. **Cron Job Cleanup** (#15427)
   - Uses job config for cleanup instead of hardcoded "keep"
   - **Important for our scheduled tasks!**

2. **QMD Memory Improvements** (#15429)
   - Uses QAT variant of embedding model for better quality
   - Default searchMode now "search" instead of hybrid
   - Should improve our memory recall accuracy

3. **replyToMode Default Changed**
   - Default changed from "first" to "off"
   - Replies only happen with explicit `[[reply_to_*]]` tags
   - Check if this affects our Telegram/Discord workflows

4. **Telegram 100-Command Limit** (#5787)
   - Enforces Telegram's command limit with warning
   - Prevents silent failures on command-heavy sessions

5. **Discord historyLimit Fix** (#11356)
   - Applies historyLimit to channel/group sessions
   - Prevents compaction bypass for our Discord agents

6. **Heartbeat Wake Exemption** (#14532)
   - Wake and hook reasons exempt from empty-heartbeat skip
   - Our scheduled tasks won't be skipped

7. **Auto-Reply Image-Only Messages** (#12352)
   - Image-only messages now reach the agent
   - Important for our media processing workflows

8. **MP3/M4A Voice Recognition** (#15438)
   - MP3 and M4A now recognized as voice-compatible audio
   - Better audio processing support
