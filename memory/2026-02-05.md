## Key Lessons Today

25. **Sub-agents are insanely productive** тАФ 6 parallel agents produced 5 major specs + 1 working system in under 2 hours. GPT-5.2 with high thinking is the sweet spot for technical specs.

26. **Model name precision matters** тАФ `anthropic/claude-sonnet-4` тЙа `anthropic/claude-sonnet-4-0`. The `-0` suffix caused a sub-agent spawn to fail. Always use exact model IDs from config.

27. **Python `.format()` vs bash templates** тАФ Never use `.format()` for bash script templates. Bash arrays (`${FOO[@]}`), curl (`%{http_code}`), and variable expansions all conflict with Python's brace syntax. Use `.replace()` instead.

28. **Test against real sites immediately** тАФ The Unbrowse MVP had a template bug that only surfaced when generating actual scripts. Quick test cycle caught it in minutes.

29. **last30days script OOMs on Railway** тАФ The Reddit+X research script gets killed (SIGKILL) in our Railway container. Likely memory limits. Workaround: use direct web_search + web_fetch instead.

30. **Security updates need coordinated token rotation** тАФ When rotating shared tokens (hooks), must update BOTH instances simultaneously. Send new token to the other agent BEFORE changing your own config, using the old token.
