# Memory Vault

> A durable, structured memory system for personal knowledge and AI conversation history.

## Overview

Building a searchable archive of personal knowledge, including exported conversations from AI assistants (ChatGPT, Claude, Grok). Uses a three-layer architecture inspired by Nat Eliason's memory system.

## Key Points

- Three-layer architecture: Knowledge Graph (PARA), Daily Notes, Tacit Knowledge
- Atomic facts with status tracking (active/superseded)
- Designed for use with Obsidian (UI) and qmd (search)
- Memory decay tiers: Hot (7 days), Warm (30 days), Cold (older)
