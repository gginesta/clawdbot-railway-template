## Lessons Learned Today
- **`openclaw doctor --fix` danger confirmed again:** Restores stale tokens from backup — always re-check Discord/Anthropic tokens after running
- **Shared secrets.json on `/data/shared/`:** Each agent needs their OWN token — sharing Molty's Anthropic token with Raphael caused empty responses
- **Empty webchat bubbles = auth failure:** When model label shows but bubble is blank, it's the Anthropic token not being read correctly
- **GLM falls back to Chinese:** If primary fails and GLM is in fallback chain, it responds in Chinese by default
- **Don't send multiple Discord commands to a broken agent:** Cascades and crashes the session
- **Raphael's trust model works correctly:** He correctly refused to apply config changes without Discord confirmation from Guillermo
