## BuzzRounds Custom Domain Fix (14:37 HKT)
- Both `tunes.buzzrounds.com` and `ydkj.buzzrounds.com` showing Railway "Not Found" error
- Root cause: SSL certificates stuck at `VALIDATING_OWNERSHIP` — certs never issued despite DNS being correct
- Fix: Deleted and re-created custom domains on Railway. Each recreation assigns a NEW CNAME target.
- Updated Namecheap DNS records multiple times as Railway kept generating new targets
- Final CNAME targets: `tunes` → `9n6r6da5.up.railway.app`, `ydkj` → `4o69dlkz.up.railway.app`
- Status: DNS propagated, waiting on Railway cert issuance (still validating as of 15:42 HKT)
- Fallback URLs work: `tunes.up.railway.app` and `ydkj.up.railway.app`
- **Lesson:** Railway custom domain cert issuance can get permanently stuck. Delete/recreate is the fix but generates new CNAME targets each time — update DNS immediately and don't keep deleting.
