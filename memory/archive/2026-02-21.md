## Lessons
60. **Discord `actions.messages: true`** must be explicitly set to enable message reading in the tool schema. Defaults alone aren't enough to expose it in the agent's tool enum.
61. **Tool schema baked per session.** Config changes to Discord actions require a NEW session to take effect. Mid-session the enum won't update.
62. **`blockStreaming` conflicts with `streamMode`** — OpenClaw skips preview stream if blockStreaming is explicitly enabled. Choose one.
63. **`streamMode` not in binary 2026.2.20** despite docs and update notes mentioning it. Wait for next binary or check PR #22111 for actual config key.
64. **Railway `variableCollectionUpsert` uses `input:` wrapper** — not top-level params.
