## Lessons
48. **Cron scripts must use venv Python** тАФ `/data/workspace/.venv/bin/python3`, not bare `python3`. System Python lacks google-auth, requests, etc.
49. **message tool params:** `message` for text, `target` for recipient, `channel` for platform. NOT `activityState`.
50. **Browser stale lock files:** If `openclaw browser start` fails with "Failed to start Chrome CDP on port 18800", check for stale `SingletonLock`/`SingletonSocket` files in `/data/.openclaw/browser/<profile>/user-data/` and `/tmp/openclaw-browser/`. Delete them and retry. Caused by Brave crashing during gateway restarts.
51. **Anthropic is a built-in provider:** No `models.providers.anthropic` block needed in config. Just `auth.profiles.anthropic:default` with `mode: "token"`. Model IDs like `anthropic/claude-sonnet-4-6` resolve via built-in definitions.
52. **Sonnet 4.6 > Opus 4.6 for most tasks:** 5x cheaper, 5x larger context (1M vs 200K), faster, wins on agentic/practical benchmarks. Opus has slight edge on hardest reasoning only. Fleet switched to Sonnet 4.6 primary Feb 20.
53. **All cron jobs switched to direct Anthropic Haiku** (`anthropic/claude-3-5-haiku-latest`) тАФ uses Max plan daily allowance instead of OpenRouter credits.
