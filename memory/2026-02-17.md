## Lesson #30
**Always check XDG paths for OpenClaw-managed QMD.** The default `qmd status` shows an empty index. Real index is at `~/.openclaw/agents/<id>/qmd/xdg-cache/qmd/index.sqlite`. Set XDG_CONFIG_HOME and XDG_CACHE_HOME before running CLI commands.
