## Major Pivot: QMD → OpenAI Builtin (20:00-21:10 HKT)

### Why We Pivoted
After hours debugging QMD hybrid search (reranker OOM at ~6GB, cmake issues, 30s timeouts), Guillermo asked to pause and evaluate. Three options presented:
- **Option A:** Pure OpenAI (drop QMD entirely)
- **Option B:** QMD backend + OpenAI search (hybrid)
- **Option C:** Full local QMD (broken)

Key discovery: `extraPaths` config is QMD-only — OpenAI builtin only indexes `MEMORY.md` + `memory/**/*.md`. This killed Option B as originally planned.

**Solution: Option A1 — OpenAI + Architect Pattern**
- Move shared vault under `memory/vault/` on Molty only
- Builtin indexer picks it up automatically (`memory/**/*.md`)
- Other agents write to `/data/shared/memory-vault/` but don't index it
- Compartmentalization: Raphael sees only Brinc, Leonardo sees only Launchpad, Molty sees everything

### Decision (20:53 HKT)
Guillermo approved Option A1. Rationale: simplicity, reliability, all 6 goals met.
